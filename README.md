# Book Recommender with LLM Dialogue Interface

**Десктопное приложение** для подбора книг. Пользователь общается с системой на естественном языке, получает персонализированные рекомендации и управляет обратной связью (лайки/дизлайки).

### Ключевые особенности
- Комбинированная метрика схожести: манхэттенское + таксономическое + булево расстояния.
- LLM-интерфейс через API с `llama3.1:8b-instruct-q4_0`.
- Поддержка фильтрации, отката действий, сохранения истории.

### Результаты
- Точность распознавания пользовательских запросов: **95%**.
- Полностью локальный инференс (конфиденциальность данных).

### Стек
`Python 3.11` `Llama3.1` `REST API` `PyQt/Tkinter`

## Консольная версия с LLM (папка `llm_recommended`)

**Интерфейс:** командная строка, диалоговый режим.  

### Возможности
- Пользователь описывает желаемую книгу **своими словами** (например: *«хочу что‑то в стиле фэнтези, но с детективной линией»*).
- LLM преобразует запрос в структурированные параметры и/или напрямую влияет на ранжирование.
- Рекомендации строятся на **комбинированной метрике**:
  - Манхэттенское расстояние
  - Таксономическое расстояние
  - Булево расстояние
- Поддержка обратной связи (`like`/`dislike`), откат действий.

### Качество
- Точность распознавания намерений пользователя — **95%** (измерено на тестовой выборке).

##  Десктопная версия на Tkinter (папка `filtered`)

**Интерфейс:** классическое оконное приложение, управление мышью.  

### Возможности
- **Ручные фильтры:** жанр, автор, год, рейтинг.
- Оценка книг — лайки/дизлайки сохраняются в профиль.
- Система использует **ту же комбинированную метрику**, но без LLM — рекомендации строятся на основе истории оценок и выбранных фильтров.
- Полностью автономная работа (не требует внешних API).
